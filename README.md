# gocapy-market-intelligence

Go Capy market intelligence plugin — standalone feeds that watch external sources and produce digest reports. Independent of the AI SDR pipeline.

## Skills

- **defense-monitor** — weekly defense/aerospace/space contract intelligence. Scrapes war.gov, USASpending, SAM.gov, and defense industry news. Outputs a markdown report + CSV + Gmail draft digest.

(Room to grow — future home for any other market/industry monitoring skills.)

## Installation

```bash
/plugins add marcella-capy/gocapy-market-intelligence
```

## Companion plugins

- [gocapy-sdr](https://github.com/marcella-capy/gocapy-sdr) — the AI SDR pipeline (research, copywriting, campaign upload, HotHawk/PlusVibe APIs).
- [gocapy-infra](https://github.com/marcella-capy/gocapy-infra) — SiteGround + Porkbun setup tools.
